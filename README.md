## Installation for this application

This applicaton is Url Shortener built by Vuk Matijevic for a test project.

-   PHP >= 8.0.2
-   Node >= 18.19.1

1. Setting up packages

-   composer install
-   npm install

2. Using url_shortener.sql for a dump or use migration and seeder in laravel project.

3. You can run project by `php artisan serve` and `npm run dev`.

Then you can visit the url shortener in this url- http://localhost:8000.
